Engagement Comments and post management

Commerce Orders, inventory and customers

Audit Client-scoped integrations

TikTok integration use case

TikTok is used only after explicit user authorization. The integration supports advertising reporting through TikTok Business and organic publishing through TikTok Content Posting.

Connect TikTok

The user starts OAuth from Integrations and authorizes either TikTok Business or TikTok organic permissions.

Import account context

The app identifies the authorized advertiser or TikTok user and keeps the token linked to the current Algoritmia client.

Review ads or prepare content

The user can review TikTok Ads reporting, or upload media, write copy and select TikTok inside the Publisher.

Confirm action

The app reads ad data or uses Content Posting API permissions according to the approved scope and user confirmation.

Data handling

  • Tokens are used only to perform actions authorized by the connected user.
  • Social accounts are stored per client workspace and are not shared across customers.
  • Users can disconnect integrations from the app and can request data deletion.
  • No TikTok credentials are requested or stored by Algoritmia.
